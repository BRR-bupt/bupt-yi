# BUPT YI
这个项目是我的毕业设计项目，一个具有基本视频剪辑功能的网站

## Demo
[网页链接](https://buptyi.xyz)

![首页截图](https://buptyi.xyz:5000/video.png)


## 🚀 Features

- 🎪 基于 **vue3** 框架开发
- ⚡ 基于 **vite** 开发，获取更快速的启动速度
- 🦾 完全使用 **typescript** 提供完备的代码类型检查
- 🔋 使用**pinia** 提供状态管理服务，极大地简化了组件间通信

## 📦 Install
```bash
pnpm install
```

## 🦄  serve
```bash
pnpm run dev
```

## 🧱  build
```bash
pnpm run build
```
